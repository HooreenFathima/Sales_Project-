# Final Project: Sales Performance Intelligence

## Project Overview
An end-to-end data analysis pipeline built in Python to evaluate retail performance across multiple regions and product lines.

## How it Works
1. **Validation**: The script checks for the existence of the CSV file before starting.
2. **Preprocessing**: Automatically removes duplicates and handles missing numerical values.
3. **Visualization**: Generates high-resolution PNG charts for categorical comparison and market share distribution.

## Key Metrics Found
- **Total Revenue**: Calculated across all active regions.
- **Top Product**: Identified the highest-grossing inventory item.
